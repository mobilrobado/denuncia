Denunciar un Móvil Robado

¿Cómo Funciona?

Este servicio permite a los usuarios denunciar el robo de teléfonos móviles mediante el ingreso del IMEI. Una vez enviada la denuncia, se notificará a las operadoras para que bloqueen el dispositivo.

  1. Ingresa tu nombre, el IMEI del dispositivo y tu correo electrónico.
  2. Haz clic en "Enviar Denuncia".
  3. Recibirás un correo electrónico confirmando que tu denuncia ha sido enviada.
  4. Las operadoras serán notificadas para que bloqueen el dispositivo.
